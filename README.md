#  ✅ Proyecto: ToDo List con HTML, CSS y JavaScript

Una **ToDo List** (lista de tareas) simple hecha desde cero usando HTML, CSS, JavaScript y el framework de estilos Bootstrap 5 sin frameworks. Permite al usuario agregar tareas, marcarlas como completadas y eliminarlas.


## 🔧 Funcionalidades

- ➕ Agregar nuevas tareas
- ✅ Marcar tareas como completadas
- 🗑️ Eliminar tareas

## 🧱 Tecnologías Usadas

- **HTML5**
- **CSS3**
- **JavaScript Vanilla**
- **Bootstrap**
